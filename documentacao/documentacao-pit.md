# Documentação do Projeto – PIT

## Visão Geral
Este projeto representa um aplicativo simples para uma loja fictícia de cupcakes. O app permite visualizar os produtos disponíveis, ver detalhes básicos e adicionar itens ao carrinho de forma simulada.

## Funcionalidades
- Exibição da lista de cupcakes
- Tela de detalhes do produto
- Carrinho simples
- Confirmação fictícia do pedido

## Tecnologias
- Flutter para o aplicativo
- Node.js para o back-end básico
- PostgreSQL para armazenar produtos e pedidos

## Estrutura do Sistema
O app se comunica com um back-end que retorna os produtos e registra pedidos simulados. O banco de dados contém apenas dados básicos das tabelas.
